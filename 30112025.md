### Learning Cloud Native Computing
- reference URL https://youtu.be/BfdOSwYotj8
- What is Cloud Native Computing?
  - Deploying applications as Microservices in Containers and dynamically orchestrating them to optimize resource use. (Key words - Deploy applications as Miroservices; Dynamic Orchestration).
  - CNCF is the steward/guardians of Cloud Native Ecosystem.
  - KCNA is the official entry point to understand the fundamentals of Kubernetes.
  - Kubernetes, Fluentd, Linkerd, Prometheus, OpenTracing, gRPC. These tools form the foundation of Cloud Native Applications.
  - Exam comprises of 4 Sections and Kubernetes is 44% of it. (Core Concepts, Admin)
  - Container Orchestration 28%. (Containerization, Networking)
  - Application Delivery 16%. (Storage, Debugging)
  - CN Architecture 12%.(Ecosystem, Community)
